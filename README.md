## What happened?

This repository contained the Agda formalisation and LaTeX manuscript for *FirstDistinction*, a project exploring the logical emergence of structure from a single primordial distinction.

The work has evolved significantly. The full, current version — including the complete proof chain, companion papers, and book manuscript — now lives in the **Void and Form** repository.

## Zenodo record

The original publication archived under this repository remains permanently available and citable at its Zenodo DOI. The **Void and Form** repository continues this lineage and will carry future releases and DOIs.
